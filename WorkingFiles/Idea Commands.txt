ctlr + c - копировать
ctlr + v - вставить
ctrl + a - копировать все
ctrl + x - вырезать

shift + f6 - Переменовать(перменную,класс,метод)

ctrl + / - закомментировать выделенный фрагмент
ctrl + shift + / - закомментировать многострочным кооментарием

ctrl + пробел - автодополнение

alt + enter = решать ошибки
